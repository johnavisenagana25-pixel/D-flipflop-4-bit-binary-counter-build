# 🚀 Flip-Flop & Counter Build using Verilog HDL

## 🎯 Objective

This project implements a D Flip-Flop and a 4-Bit Binary Counter using Verilog HDL.

---

## 🛠️ Tools Used

* ✅ Verilog HDL
* ✅ EDA Playground
* ✅ Icarus Verilog
* ✅ EPWave

---

## 📖 Project Description

The D Flip-Flop stores 1-bit data based on clock signals. The 4-Bit Counter increments its value on every positive edge of the clock pulse.

This project demonstrates:

* ✔️ Sequential logic circuits
* ✔️ Clock-based operations
* ✔️ Binary counting
